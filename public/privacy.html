1. 수집하는 개인정보

로또AI(com.mrrobot.lottoai)는 서비스 제공을 위해 다음 정보를 처리합니다:

  • 저장 번호 데이터: 기기 내부 저장소(localStorage)에만 저장, 외부 서버 전송 없음
  • Claude API 키: 기기 내부 저장소에만 저장, 외부 전송 없음
  • 당첨번호 데이터: 동행복권 공식 API에서 수신 (읽기 전용)

2. 개인정보 수집·이용 목적

  • 로또 번호 예측 서비스 제공
  • 번호 저장 및 히스토리 관리
  • 당첨번호 크롤링 및 통계 분석

3. 개인정보 보유 및 이용 기간

모든 데이터는 사용자 기기에만 저장되며, 앱 삭제 시 자동으로 삭제됩니다. 외부 서버에 개인정보를 전송하거나 저장하지 않습니다.

4. 제3자 제공

수집한 개인정보를 제3자에게 제공하지 않습니다.

5. 외부 서비스

  • 동행복권 API (www.dhlottery.co.kr): 당첨번호 조회 목적
